✕² √ Potęgi i Pierwiastki
✕²

Co to jest potęga?

Podstawowa definicja i budowa potęgi

an = a · a · a · ... · a
(a mnożone przez siebie n razy)
a – podstawa (co mnożymy)
n – wykładnik (ile razy)
wynik – wartość potęgi
Przykład podstawowy
2³ = 2 · 2 · 2
= 8
Dwa mnożymy przez siebie 3 razy
Każda liczba do potęgi 0
a⁰ = 1
7⁰ = 1, 100⁰ = 1, 999⁰ = 1
Wyjątek: 0⁰ jest nieokreślone!
Każda liczba do potęgi 1
a¹ = a
5¹ = 5, 13¹ = 13
Dowolna liczba do pierwszej potęgi = ta sama liczba
Zero do dowolnej potęgi
0ⁿ = 0 (n > 0)
0⁵ = 0, 0¹⁰⁰ = 0
Zero do każdej dodatniej potęgi = 0
⚡

Prawa działań na potęgach

Najważniejsze wzory – musisz je znać na pamięć!

🔹 Mnożenie – ta sama podstawa
aᵐ · aⁿ = aᵐ⁺ⁿ
2³ · 2⁴ = 2⁷ = 128
Podstawy muszą być takie same. Wykładniki dodajemy!
🔹 Dzielenie – ta sama podstawa
aᵐ ÷ aⁿ = aᵐ⁻ⁿ
3⁵ ÷ 3² = 3³ = 27
Podstawy takie same. Wykładniki odejmujemy!
🔹 Potęga potęgi
(aᵐ)ⁿ = aᵐ·ⁿ
(2³)⁴ = 2¹² = 4096
Potęga do potęgi – wykładniki mnożymy!
🔹 Iloczyn do potęgi
(a · b)ⁿ = aⁿ · bⁿ
(2 · 3)⁴ = 2⁴ · 3⁴ = 16 · 81
Każdy czynnik podnosimy osobno do tej potęgi
🔹 Iloraz do potęgi
(a ÷ b)ⁿ = aⁿ ÷ bⁿ
(6 ÷ 2)³ = 6³ ÷ 2³ = 216 ÷ 8 = 27
Licznik i mianownik podnosimy osobno
🔹 Różne podstawy – ta sama potęga
aⁿ · bⁿ = (a · b)ⁿ
2³ · 5³ = 10³ = 1000
To odwrotność reguły iloczynu do potęgi
⚠️ Uwaga! Wzory z mnożeniem/dzieleniem działają tylko gdy podstawy są takie same (np. 2³ · 2⁴) LUB gdy wykładniki są takie same (np. 2³ · 5³). Nie możesz tak uprościć np. 2³ · 3⁴!
⁻¹

Wykładniki ujemne i ułamkowe

Potęgi które „odwracają" lub „pierwiastkują"

Wykładnik ujemny
a⁻ⁿ = 1 / aⁿ
2⁻³ = 1/2³ = 1/8 = 0,125
Ujemny wykładnik = odwróć ułamek! Podstawa idzie „do mianownika".
Wykładnik −1
a⁻¹ = 1/a
5⁻¹ = 0,2   4⁻¹ = 0,25   10⁻¹ = 0,1
Odwrotność danej liczby
Wykładnik ½ = pierwiastek
a^(1/2) = √a
9^(1/2) = √9 = 3
Połowa wykładnika = pierwiastek kwadratowy
Wykładnik ⅓ = sześcienny
a^(1/3) = ∛a
27^(1/3) = ∛27 = 3
Jedna trzecia wykładnika = pierwiastek sześcienny
Ułamkowy wykładnik ogólnie
a^(m/n) = ⁿ√(aᵐ)
8^(2/3) = ∛(8²) = ∛64 = 4
Licznik = potęga, mianownik = stopień pierwiastka
Potęgi dziesiętne (10⁻ⁿ)
10⁻ⁿ = 0,00...01
10⁻¹=0,1   10⁻²=0,01   10⁻³=0,001
n-ta potęga ujemna liczby 10 = n miejsc po przecinku
−a

Potęgi z ujemną podstawą

Tu najczęściej są błędy na sprawdzianie!

✅ Parzysta potęga = PLUS
(-3)² = (-3) · (-3) = +9
(-2)⁴ = +16
Ujemna liczba do parzystej potęgi zawsze daje wynik DODATNI.
⚠️ Nieparzysta potęga = MINUS
(-3)³ = (-3)·(-3)·(-3) = −27
(-2)⁵ = −32
Ujemna liczba do nieparzystej potęgi zawsze daje wynik UJEMNY.
🚨 PUŁAPKA: −a² to nie (−a)²!
−3² = −(3·3) = −9
(−3)² = (−3)·(−3) = +9
Minus BEZ nawiasu należy do całego wyrażenia, nie do podstawy!
ℹ️ Jedynka i minus jedynka
1ⁿ = 1 (zawsze)
(−1)parzyste = 1  |  (−1)nieparzyste = −1
Zapamiętaj: (−1)¹⁰⁰ = 1, (−1)⁹⁹ = −1
√

Pierwiastki – definicja i działania

Pierwiastek odwraca potęgowanie

Pierwiastek kwadratowy
√a = b ⟺ b² = a
Wynik zawsze ≥ 0. Np. √9 = 3 (nie −3!)
Pierwiastek sześcienny
∛a = b ⟺ b³ = a
Wynik może być ujemny! Np. ∛(−8) = −2
Mnożenie pierwiastków
√a · √b = √(a · b)
√2 · √8 = √16 = 4
Pierwiastki tego samego stopnia można połączyć
Dzielenie pierwiastków
√a ÷ √b = √(a ÷ b)
√18 ÷ √2 = √9 = 3
Podobnie jak mnożenie, ale dzielimy pod pierwiastkiem
Upraszczanie pierwiastka
√(a²·b) = a·√b
√12 = √(4·3) = 2√3
Wyciągamy pełne kwadraty przed pierwiastek
Pierwiastek z kwadratu
√(a²) = |a|
√((-5)²) = √25 = 5, nie −5!
Pierwiastek kwadratowy ZAWSZE jest nieujemny
Podniesienie do kwadratu
(√a)² = a
(√7)² = 7   (√13)² = 13
Kwadrat pierwiastka „znosi" pierwiastek
Brak pierwiastka z ujemnej
√(−a) = brak w ℝ
√(−4) – nie istnieje w liczbach rzeczywistych
Nie da się wyciągnąć √ z liczby ujemnej (w szkole podstawowej)

Jak rozwiązywać zadania – krok po kroku

Gotowe przepisy na typowe zadania

📌 Upraszczanie iloczynu potęg (np. 2³ · 2⁴)
1
Sprawdź czy podstawy są takie same (tu: obie 2) ✓
2
Dodaj wykładniki: 3 + 4 = 7
3
Wynik: 2⁷ = 128
📌 Upraszczanie pierwiastka (np. √72)
1
Rozłóż na czynniki z pełnym kwadratem: 72 = 36 · 2
2
Rozdziel: √(36 · 2) = √36 · √2
3
Oblicz: √36 = 6, wynik: 6√2
📌 Potęga ułamkowa (np. 27^(2/3))
1
Mianownik = stopień pierwiastka: ∛27
2
Oblicz pierwiastek: ∛27 = 3
3
Podnieś do licznika: 3² = 9
📌 Potęga ujemna (np. 2⁻⁴)
1
Ujemny wykładnik = odwrotność
2
Zapisz: 2⁻⁴ = 1/2⁴
3
Oblicz: 1/16 = 0,0625
