📖 Teoria – Potęgi i Pierwiastki

Podstawy Czym jest potęga?

aⁿ = a · a · a · ... · a (n razy)
a – podstawa potęgi, n – wykładnik
2³ = 8
3⁴ = 81
5² = 25
10³ = 1000
Uwaga: Każda liczba podniesiona do potęgi 0 równa się 1, np. 7⁰ = 1, 100⁰ = 1

Działania Mnożenie i dzielenie

WzórPostaćPrzykład
Mnożenie (ta sama podstawa)aᵐ · aⁿ = aᵐ⁺ⁿ2³ · 2⁴ = 2⁷
Dzielenie (ta sama podstawa)aᵐ ÷ aⁿ = aᵐ⁻ⁿ3⁵ ÷ 3² = 3³
Potęga potęgi(aᵐ)ⁿ = aᵐ·ⁿ(2³)⁴ = 2¹²
Iloczyn do potęgi(a·b)ⁿ = aⁿ · bⁿ(2·3)⁴ = 2⁴ · 3⁴
Iloraz do potęgi(a÷b)ⁿ = aⁿ ÷ bⁿ(6÷2)³ = 6³ ÷ 2³

Wykładnik Potęgi ujemne i ułamkowe

Wykładnik ujemny:
a⁻ⁿ = 1 / aⁿ
Przykład:
2⁻³ = 1/8 = 0,125
Wykładnik ½:
a^(1/2) = √a
Wykładnik ⅓:
a^(1/3) = ∛a
Zapamiętaj: a⁻¹ = 1/a, np. 5⁻¹ = 0,2 ; a⁰ = 1 (dla a ≠ 0)

Pierwiastki Definicja pierwiastka

√a = b ⟺ b² = a (b ≥ 0)
Pierwiastek kwadratowy z a to liczba b ≥ 0, której kwadrat równa się a
∛a = b ⟺ b³ = a
Pierwiastek sześcienny – wynik może być ujemny!
√4 = 2
√9 = 3
√25 = 5
√100 = 10
∛8 = 2
∛27 = 3

Ważne! Szczególne przypadki i pułapki

(-a)² = a² ✓ (wynik dodatni) (-3)² = 9
Ujemna liczba do parzystej potęgi = wynik dodatni
(-a)³ = -a³ (wynik ujemny) (-3)³ = -27
Ujemna liczba do nieparzystej potęgi = wynik ujemny
-a² ≠ (-a)² -3² = -9, (-3)² = 9
Minus przed potęgą to minus przed całym wynikiem!
√(a²) = |a| (wartość bezwzgl.) √((-3)²) = √9 = 3, nie -3
Pierwiastek kwadratowy jest zawsze ≥ 0
0ⁿ = 0 (n > 0) 0⁰ – wartość nieokreślona a⁰ = 1 (dla a ≠ 0)
Zera i jedynki jako podstawy
1ⁿ = 1 (każda potęga 1) (-1)parzysta = 1 (-1)nieparzysta = -1
Podstawa 1 lub -1 – specjalne zasady
